# macOS Web 🖥️

一个用纯 HTML / CSS / 原生 JavaScript 实现的 macOS 风格网页操作系统。无依赖、无构建步骤,双击 `index.html` 即可运行。

## 运行

```bash
open index.html        # macOS 下直接打开
# 或任意静态服务器
python3 -m http.server 8000
```

## 部署

线上地址:**https://desktop.19ba.cn**(阿里云 ECS + Caddy,Let's Encrypt 自动 HTTPS)。

改完代码后一条命令发布(rsync 同步到服务器,Caddy 即时生效,并自动做健康检查):

```bash
./deploy.sh
```

服务器别名、远程目录、域名等配置在 `deploy.sh` 顶部。

## 功能

**系统**
- 开机动画(Apple 标志 + 进度条),菜单栏苹果菜单里可以睡眠 / 重启 / 关机
- 顶部菜单栏:实时时钟、电池、Wi-Fi,随当前应用切换的下拉菜单(文件 / 编辑 / 显示 / 窗口…)
- Dock:鼠标靠近时图标放大、启动弹跳动画、运行小圆点、悬停名称气泡
- 窗口管理:拖动、八方向缩放、红绿灯(关闭 / 最小化到 Dock / 缩放)、双击标题栏最大化、焦点层级
- 聚焦搜索(⌘K 或菜单栏放大镜):搜应用,还能直接算算术
- 启动台、控制中心(深色模式、亮度、假装的 Wi-Fi / 蓝牙)、通知横幅
- 桌面右键菜单:新建文件夹、更改墙纸、切换深浅色
- 深色模式、7 张内置壁纸、强调色、Dock 大小,全部记忆在 localStorage

**内置应用**
| 应用 | 说明 |
|---|---|
| 访达 | 虚拟文件系统,侧边栏导航、前进后退、双击打开文件 |
| Safari | 真实 iframe 浏览器 + 起始页收藏夹 |
| 信息 | 给 Claude / 老妈 / 王小明 发消息,会自动回复 |
| 备忘录 | 多条笔记,自动保存到本地 |
| 日历 | 月视图,可翻页回到今天 |
| 音乐 | 假装在播放,进度条和旋转唱片是真的 |
| 计算器 | 完整可用的四则运算 |
| 终端 | ls / cd / cat / open / neofetch / sudo 等命令 |
| App Store | 浏览 / 安装 / 卸载 / 发布网页应用(见下) |
| 系统设置 | 外观、墙纸、程序坞设置 |

## App Store

- **商店**:内置 8 个可安装应用(贪吃蛇、2048、记忆翻牌、书阁电子书、画板、番茄钟、秒表、世界地图),有发现页、分类、搜索、详情页和下载动画
- **书阁**:一款界面考究的电子书阅读器——CSS 多栏真实分页 + 手势跟手翻页,内置《道德经》《唐诗选》《世说新语》及原创《本机手册》;支持目录跳转、纸/米黄/夜间三主题、字号行距调节
- **安装**:点"获取"即装。已安装应用出现在启动台、聚焦搜索和访达"应用程序"目录,持久化在 localStorage,刷新自动恢复;商店里随时可卸载
- **运行**:HTML 应用跑在 `sandbox="allow-scripts"` 的 iframe 里,与系统完全隔离;URL 应用以普通 iframe 加载外部网页
- **发布**:侧边栏"发布应用"——填名称、emoji 图标、分类、简介,贴一段 HTML 代码(或填一个网址)即可上架。发布后与内置应用一样可安装、打开、卸载、下架

## 结构

```
index.html        页面骨架
style.css         全部样式(含深色模式)
js/wm.js          窗口管理器
js/apps.js        应用注册表 + 系统应用实现 + 虚拟文件系统
js/store.js       App Store(商店 UI / 安装 / 发布 / 沙箱运行器)
js/system.js      菜单栏 / Dock / 聚焦 / 启动台 / 控制中心 / 通知
js/main.js        入口
apps/catalog.js   商店目录清单(应用元数据)
apps/<id>/        商店应用本体,每个一个独立目录(沙箱 iframe 按需加载)
```

### 给商店新增应用

1. 在 `apps/你的应用/` 下写一个自包含的 `index.html`(可直接双击调试)
2. 在 `apps/catalog.js` 加一条记录(`type: 'page'`, `source` 指向该文件)

应用源码按需加载:用户不点开就不下载,与系统运行在隔离沙箱中。
